# Skill Demo

Demonstrates loading Claude Code SKILL.md files into Semantic Kernel.

See [`samples/SkillDemo/Program.cs`](https://github.com/JerrettDavis/JD.SemanticKernel.Extensions/tree/main/samples/SkillDemo) for the full source.

## What it does

1. Creates a sample SKILL.md file with YAML frontmatter
2. Loads it using `UseSkills()` extension method
3. Displays the registered SK functions and their metadata
